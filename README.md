# km-porn-spider

#### Description
##### Crawler for 快猫短视频
##### If the endpoint cannot access, just change it in  `constatn.py` 


#### Environment 
- python3+

#### Install dependencies
```shell
pip install requests
pip install json
#for mac use: pip install pycryptodome
pip install pycrypto
```

#### Run script
```shell
#crawl hot videos
python main.py 
#crawl all videos
python main.py 1
```
